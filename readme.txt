Mi proyecto es un modelo de compra/venta de piezas de arte. 

MODELOS:
-Pieza:
Nombre (nombre de tu pieza de arte)
Descripcion (si se tuvo alguna inspiracion, o si tiene algun significado tu pieza)
Tecnica (la forma o materiales con la cual se realizo la pieza ejemplo: oleo, grafito, acuarela, etc)
Dimensiones (las dimensiones se refiere al perimetro del lienzo ejemplo: 20cmX35cm)
Precio

-Comprador
usuario
nombre
direccion (direccion de envio de tu compra)
correoElectronico
numeroTelefono

-Vendedor
usuario
nombre
correo electronico 
numero de telefono



Por el momento el programa puede registrar usuarios de compra, y de venta. Puede registrar tu pieza de arte para vender; se encuentra el catalogo de las piezas registradas,
y se puede buscar en base a la tecnica que gustes, las piezas de arte que existen dentro del catalogo.
